# Visualización de datos de Salud de España

Esta visualización hace parte del Trabajo de Fin de Master **"Herramienta interactiva para la visualización y predicción de la relación entre la alimentación y la salud en España a partir de datos abiertos"**.

En el proyecto se desarrolla una aplicaciíon usando como **Streamlite**. Todo el desarrollo se realiza en linea.

La aplicacion permite:

  1. Visualizar análisis descriptivo del conjunto de datos abiertos de la Encuesta de Salud de España 2023.
  2. Realizar predicciones usando um modelo de entrenado y evaluado

Se incluyen los archivos del dataset ya procesado y el modelo final entrenado.
